Hugo + Github pages 建立个人网页的方法

Feb 28, 2021 1 min read

安装macports

安装过程中，会停止，按option-command+esc强制关闭。

升级macports sudo port -v selfupdate 出错，修改~\opt\local\etc\macports\sources.conf

将最后一句用#注释掉， rsync://rsync.macports.org/macports/release/tarballs/ports.tar [default] 并换成 https://distfiles.macports.org/ports.tar.gz [default]

然后再运行 sudo port -v selfupdate可以升级

安装Hugo

在Terminal安装Hugo，会提示安装go

sudo port install hugo

想缷载可以使用： sudo port uninstall hugo

安装好后，到https://github.com/gohugoio/hugo/releases下载hugo_extended_0.81.0_macOS-64bit.tar.gz14.2 MB

解压后，将其中的hugo应用文件拷贝到\opt\local\bin里，进行替换，改成extended版本

运行hugo，成功建站

过程参考： https://leidawt.github.io/en/post/借助hugo和academic主题在github/

  • git add .

  • git commit -m “Initial commit”

  • git push -u origin master

  • hugo

  • cd public

  • git add .

  • git commit -m “Build website”

  • git push origin master

  • cd ..

发布新文档方法：

  • 打开Terminal
  • 进入myweb
  • hugo new –kind post post/test #在myweb/content/post下会建立一个test文件夹，里面test.md可以修改会新的post

修改完后，运行hugo进行建站

  • 进入cd public
  • git add .
  • git commit -m “new”
  • git push origin master
